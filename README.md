## 导入数据库
数据库文件保存在Data文件夹下，导入即可.

## 修改数据库连接配置

修改`includes/common.inc.php`：

```php
define('DB_HOST', 'localhost');  //地址
define('DB_USER', 'root');       //用户名
define('DB_PWD', '');            //密码
define('DB_NAME', 'gmanage');    //数据库名
```
